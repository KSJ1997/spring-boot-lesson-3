GET /books - получить список всех книг
GET /books/{id} - получить информацию о книге по ID
POST /books - создать новую книгу
PUT /books/{id} - обновить информацию о книге по ID
DELETE /books/{id} - удалить книгу по ID

GET /readers - получить список всех читателей
GET /readers/{id} - получить информацию о читателе по ID
POST /readers - создать нового читателя
PUT /readers/{id} - обновить информацию о читателе по ID
DELETE /readers/{id} - удалить читателя по ID

GET /issues - получить список всех выдач книг
GET /issues/{id} - получить информацию о выдаче по ID
POST /issues - создать новую выдачу
PUT /issues/{id} - обновить информацию о выдаче по ID
DELETE /issues/{id} - удалить выдачу по ID
